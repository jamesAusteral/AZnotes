# 我的笔记

这里是我的个人笔记站：课程笔记、实验踩坑记录、科研日志，以及每天读到的东西。

!!! note "这个站点为什么这样设计"
    - 我用中文书写，重在**记录**而不是排版。
    - 笔记仓库是**私有**的（源码不公开），但站点本身**公开**，因为我希望笔记被更多人看到。
    - 看不懂的专有名词，会在下面放一个可展开的解释块。
    - 实验记录是这里最大的部分，重点是"我踩过的坑"。

## 网站想法记录

把脑子里关于这个站的想法随手记在这里，想到就加：

- 课程笔记 + 实验记录分开，实验记录按"问题"组织而不是按时间。
- 科研进度（LLM Wiki）单独一块，可延拓，每次尝试打勾 + 记时间。
- 日志区放科研日记、开发日志、每日阅读。
